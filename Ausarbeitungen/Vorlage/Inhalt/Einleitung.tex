Ein Zitat zum Erstellen der Bibliographie: \Zitati[S.~35]{Martin2008}.
Es werden nur die verwendeten Quellen im Literaturverzeichnis aufgeführt, sodass
eine zentrale Datei für alle Ausarbeitungen genutzt werden.

Nur verwendete Abkürzungen (wie z.B. \ac{HTML}) werden im Abkürzungsverzeichnis aufgeführt.
Daher kann eine zentrale Datei für alle Ausarbeitungen genutzt werden.

Verschiedene Verweise: Abbildung \ref{fig:Beispiel}: \nameref{fig:Beispiel} auf Seite \pageref{fig:Beispiel}, Tabelle \ref{tab:Beispiel}: \nameref{tab:Beispiel} auf Seite \pageref{tab:Beispiel}, Listing \ref{lst:Beispiel}: \nameref{lst:Beispiel} auf Seite \pageref{lst:Beispiel}.
